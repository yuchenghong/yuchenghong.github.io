发表于 2017-12-24 | 分类于 nosql

数据类型

Redis支持五种数据类型：string（字符串），hash（哈希），list（列表），set（集合）及zset(sorted set：有序集合)。

Redis命令

执行redis命令必须先要启动客户端。

Redis key说明

可以用任何二进制的序列作为Key, 比如”key”,或者二进制图片的内容，都能作为key。
比较好的习惯是通过冒号:，把Key分成几段。如
user:zhangsan:name
